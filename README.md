# SMSActivate





## Requirements

Firebase Firestore and Real Time Database have to be activated.

## Installation

To install it, simply add the following line to your Podfile:

```ruby
pod 'SMSActivate', :git => 'https://github.com/alperengezgin/smsactivate.git'
```
